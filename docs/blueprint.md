# **App Name**: WealthPath Calculator

## Core Features:

- Input Fields: Provide input fields for initial investment, monthly contributions, annual interest rate, and investment duration.
- Future Value Calculation: Calculate the projected future portfolio value based on the input parameters and compound interest.
- Graphical Projection: Generate an interactive Acorns-style graph showing the growth of the investment over time, visualizing both the total investment and the returns. Add tool tips.
- Annual Breakdown Display: Present a dropdown to display a year-by-year breakdown of the investment growth, highlighting contributions and returns for each year.
- Account Types: Add account types Roth IRA and Traditional IRA.

## Style Guidelines:

- Primary color: Use a deep, muted green (#2E4A31) to evoke a sense of growth and stability.
- Background color: Employ a very dark, desaturated grey (#121214) in keeping with a professional dark theme.
- Accent color: Use a bright, contrasting green (#96E559) for highlighting key information and interactive elements.
- Body and headline font: 'Inter', a modern grotesque-style sans-serif, will be used for all body and headline text to keep a consistent design aesthetic.
- Use glass card styling for input fields and data displays, providing a frosted glass appearance with subtle transparency.
- Implement clean, crisp SVG icons for visual cues, maintaining consistency and scalability across devices.
